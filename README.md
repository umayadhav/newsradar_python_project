NewsRadar is a personal news intelligence web app that fetches live news 
from 70,000+ sources worldwide using the NewsAPI. Built using core Python OOP 
principles and Django, it allows users to browse news by category, search 
by keyword, save favourite articles, and generate a daily digest.

This project was built to demonstrate:
- Clean Python OOP architecture (Classes, Inheritance, Encapsulation, Polymorphism)
- Real-world REST API integration
- Full-stack Django web development
- Modular, production-style project structure

---

 ✨ Features

| Feature | Description |
|---|---|
| 🌍 Top Headlines | Live top headlines from BBC, CNN, Reuters |
| 💻 Technology News | Latest tech news fetched in real-time |
| 🏏 Sports News | Cricket, IPL, Football and more |
| 💰 Business News | Markets, economy, finance updates |
| 🔍 Keyword Search | Search any topic from 70,000+ sources |
| ⭐ Favourites | Save and manage favourite articles |
| 📋 Daily Digest | Top 5 headlines with file export |
| 🌙 Dark UI | Modern dark-themed responsive design |
